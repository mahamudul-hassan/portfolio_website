# Fahimul Haque — Academic Portfolio

Personal research portfolio for Fahimul Haque, IPE undergraduate at BUET.

## Stack
Vanilla HTML / CSS / JS · Vercel Serverless Functions · Anthropic Claude API

## Setup
1. Add `formal_pic_fahim.png` and `CV.pdf` to the root folder
2. Deploy to Vercel, set `ANTHROPIC_API_KEY` environment variable
3. Done!

## Environment Variables
| Variable | Description |
|---|---|
| `ANTHROPIC_API_KEY` | Anthropic API key for the chatbot |
